A comprehensive WordPress plugin that analyzes and tracks Elementor widget usage across all content types in your WordPress site.

== Description ==

This plugin provides a comprehensive analysis of Elementor widget usage on your WordPress website. It helps you understand which widgets are most frequently used, where they are located, and provides insights into your overall page building patterns.

Key Features:

*   **Widget Usage Analysis:** Get a detailed breakdown of every Elementor widget used across your site.
*   **Usage Counts:** See how many times each widget is used.
*   **Content-Specific Analysis:** Filter widget usage by post type (e.g., pages, posts, custom post types).
*   **Visual Charts:** View interactive charts that visualize widget distribution.
*   **Dashboard Widget:** See a summary of widget usage directly on your WordPress dashboard.

Whether you're a developer looking to optimize your site, or a site owner curious about your content, Widgets Analyzer for Elementor provides the tools you need to gain valuable insights into your Elementor-built pages.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/widgets-analyzer-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Navigate to the 'Elementor > Widgets Analyzer' menu in your WordPress admin to view the analysis.

== Frequently Asked Questions ==

= Does this plugin work with Elementor Pro? =

Yes, the plugin analyzes usage for both core Elementor widgets and Elementor Pro widgets.

= Will this plugin slow down my site? =

The analysis is done in the WordPress admin and does not impact the performance of your live site for visitors.
